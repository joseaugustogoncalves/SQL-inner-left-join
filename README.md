# Introdução às Bases de Dados Relacionais e SQL

Este repositório serve de apoio à introdução dos conceitos fundamentais relacionados com **bases de dados relacionais** e à sua aplicação prática em SQL.

## Conteúdos Abordados

- Conceito de base de dados relacional  
- Identificação de tabelas  
- Chaves primárias (Primary Keys)  
- Chaves estrangeiras (Foreign Keys)  
- Relações entre tabelas  

## Junções em SQL

É feito o enquadramento da importância das **junções (JOINs)** em SQL, destacando o seu papel essencial na análise, exploração e cruzamento de dados provenientes de múltiplas tabelas.

## Exercício Prático

Introdução do exercício inicial e apresentação da **base de dados de suporte**, que será utilizada ao longo da aula para aplicação prática dos conceitos abordados.

Este repositório funciona como base de trabalho para exploração, análise e consolidação dos conteúdos lecionados.


## Sumário

Análise de uma base de dados relacional a partir de um exercício introdutório, com identificação de tabelas, chaves primárias, chaves estrangeiras
e tipos de relações existentes. Revisão do conceito e utilização do INNER JOIN na combinação de dados entre tabelas relacionadas. Exploração
aprofundada do LEFT JOIN, analisando o seu comportamento, diferenças relativamente ao INNER JOIN e situações práticas de utilização.
Resolução de exercícios aplicados ao LEFT JOIN e aplicação dos conceitos à base de dados do projeto da PAP.
